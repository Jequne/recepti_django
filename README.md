# Каталог кулинарных рецептов (Django)

## Быстрый старт

### 1. Клонируйте проект или скачайте архив

### 2. Откройте терминал в папке проекта

### 3. Создайте и активируйте виртуальное окружение

**Windows:**
```sh
python -m venv venv
venv\Scripts\activate
```
или (PowerShell):
```sh
.\venv\Scripts\Activate.ps1
```

**Linux/Mac:**
```sh
python3 -m venv venv
source venv/bin/activate
```

### 4. Установите зависимости
```sh
pip install django pillow
```
или, если есть файл requirements.txt:
```sh
pip install -r requirements.txt
```

### 5. Примените миграции
```sh
python manage.py migrate
```

### 6. (Опционально) Заполните тестовыми данными
```sh
python manage.py fill_test_data
```

### 7. Создайте суперпользователя для админки
```sh
python manage.py createsuperuser
```

### 8. Запустите сервер
```sh
python manage.py runserver
```

### 9. Откройте сайт в браузере
- Главная: http://127.0.0.1:8000/
- Регистрация: http://127.0.0.1:8000/register/
- Вход: http://127.0.0.1:8000/login/
- Админка: http://127.0.0.1:8000/admin/

---

## Возможности админки
- Добавление, редактирование, удаление рецептов, комментариев, оценок
- Управление пользователями
- Модерация контента

---

**Примечания:**
- Для загрузки изображений убедитесь, что папка `media/` существует (создайте вручную, если нужно)
- Все команды выполняйте в активированном виртуальном окружении
- Для выхода из виртуального окружения используйте команду `deactivate`
